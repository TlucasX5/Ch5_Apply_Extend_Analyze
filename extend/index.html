Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

You need to create a style rule that will apply two positions to accomodate Safari and all other browsers as well as a top property of zero.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

Used top 150px, left 60px. Moved the box down and to the left. Moves the element around the default position without affecting others. The other boxes remaind in their original position while this box moved and left white space

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

Used top 0px, right 0px. Only repsects the ratio of the viewport. Moved to upper right corner.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

Used bottom 90px and right 20px. Removed from the flow of the document--like it's not there. The other boxed adjusted and took up this box place like its was never there even though its still showing.
